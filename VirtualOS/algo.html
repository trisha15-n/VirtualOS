ALGORITHM

  1. Input: Get a list of pages and the number of available frames (memory slots).
  2. Initialization: Use a queue to manage the pages in memory (FIFO order). Initialize a page fault counter.
  3. Page Processing:
    • If the page is in memory, do nothing.
    • If the page is not in memory (page fault) :
      • If space is available, add the page to memory.
      • If memory is full, remove the oldest page (from the front of the queue) and load the new page.
    • Increase the page fault count for every new page loaded.
  4. Output: After processing all pages, return the total number of page faults.
